# openclaw-insight-skills

Unified repository for OpenClaw daily insight skills.

## Purpose
This repo is the consolidation target for daily insight skill families that currently live across multiple repositories.

## Planned structure
- `skills/tech-insight`
- `skills/business-insight`
- `skills/life-insight`
- `skills/humanities-insight`
- `shared/` common prompt templates, schemas, and utilities

## Migration plan
Current candidate repositories for consolidation:
- `openclaw-daily-insight-skill`
- `daily-insight`
- `daily-tech-insight`
- `daily-growth-skill`
- `daily-workplace-insights`

## Principle
Unify by product line, not by one-topic-one-repo sprawl.
