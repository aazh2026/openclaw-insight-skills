# 每日精进模板参考

## 标准每日精进模板

```markdown
# 📚 每日精进 - {{DATE}}

> 今日主题：[一句话概括今日核心]

## 一、今日核心收获

### 1. [主题名称]
- **来源**：[对话/阅读/思考/事件]
- **核心要点**：[一句话]
- **详细阐述**：[2-3段深入说明]
- **立即行动**：[具体可执行的一步]
- **验证标准**：[如何知道这招有效]

### 2. [主题名称]
...

## 二、🧠 认知突破

### 突破 1：[突破标题]
| 维度 | 内容 |
|------|------|
| **领域** | [投资/管理/技术等] |
| **旧认知** | [之前的看法] |
| **新认知** | [更新后的看法] |
| **触发源** | [什么引发了改变] |
| **可信度** | [高/中/低，理由] |

## 三、🔗 跨域连接

- **[领域A] × [领域B]**：[连接描述]
  - 相似性：[共同原理]
  - 差异性：[关键区别]
  - 应用启发：[如何应用]

## 四、📋 行动清单

### 今日完成
- [x] [行动1]
- [x] [行动2]

### 明日待办
- [ ] [行动1] - 优先级：高
- [ ] [行动2] - 优先级：中

### 本周重点
- [ ] [长期行动]

## 五、💡 明日思考

[留给明天的开放性问题或探索方向]

## 六、📊 今日数据

- **学习时长**：[X小时]
- **阅读字数**：[X字]
- **输出字数**：[X字]
- **认知突破数**：[X个]
- **心情指数**：[1-10]

---

*记录时间：{{TIME}}*
*记录者：[Agent名称]*
```
